一款基于 Flutter 开发的现代化、高性能移动端 Markdown 浏览器。它不仅是阅读器，更是您灵感的记录地。

✨ 产品特性
🚀 沉浸式阅读：默认开启预设文档，打开即读，无干扰体验。

📂 自由存取：深度集成系统文件选择器，支持手机本地任何 .md 文件的快速检索与展示。

✍️ 灵动编辑：支持实时编辑功能，排版简洁优雅，让移动端创作不再局促。

🖼️ 桌面组件：配备精美桌面小组件（Widgets），关键信息无需跳转，直达主屏。

🌓 全景适配：

暗夜模式：深度适配视觉保护，提供丝滑的深色主题切换。

自适应布局：完美支持横竖屏自由切换，无论办公还是阅读皆顺手。

🎨 视觉哲学
项目秉持 Modern & Minimalist 的设计理念：

极简主义：去除繁琐的 UI 堆砌，让文字成为主角。

动效美学：基于 Material Design 3 规范，提供华丽的微动效与流畅的页面过渡。

🛠️ 技术栈
Engine: Flutter

IDE: Android Studio

Language: Dart

Platform: Android (Optimized)
