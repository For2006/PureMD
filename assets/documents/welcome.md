# PureMD

> 一款 **Modern & Minimalist** 的移动端 Markdown 编辑器  
> 版本 1.0.0

沉浸于文字，记录每一次灵感闪光。

---

## 快速上手

打开任意 Markdown 文件即可开始阅读。编辑器支持所见即所得的块编辑模式，让你在手机上也能轻松创作。

**阅读操作**
- 单击页面中央 → 进入/退出专注模式
- 点击右上角图标 → 分享、导出或打开设置
- 底部进度条 → 显示当前阅读位置

**编辑操作**
- 点击块左侧图标 → 切换块类型（标题、引用、代码等）
- 输入 `/` → 快速切换块类型
- 使用快捷工具栏 → 一键插入 Markdown 标记

---

## 功能概览

| 功能 | 说明 |
|------|------|
| 专注模式 | 一键隐藏界面元素，只留文字与背景 |
| 块编辑器 | 按块组织内容，灵活调整结构 |
| 最近文件 | 自动记录打开历史，快速返回 |
| 主题系统 | 亮色/暗色/羊皮纸/暖棕/冷灰/OLED 多种配色 |
| 自定义字体 | 选择你喜欢的字体与字号 |
| 导出分享 | 支持导出为 PNG、PDF、DOCX |
| 主屏幕组件 | 快速笔记与最近文件小组件 |
| Markdown 指南 | 内置完整语法参考 |

---

## 支持的格式

- `.md` / `.markdown` — Markdown 源文件
- `.txt` — 纯文本文件

---

## 关于

PureMD 是使用 Flutter 构建的开源 Markdown 编辑器，注重隐私与本地优先。

所有文件仅存储在本地，无网络请求，无数据收集。
