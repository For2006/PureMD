# 欢迎使用 PureMD

> 一款 Modern & Minimalist 的移动端 Markdown 阅读器

## 功能特性

### 沉浸式阅读
- **专注模式**：一键隐藏所有界面元素，只留下清晰的文字与柔和的背景
- **阅读进度**：底部进度条实时显示阅读位置

### 自由存取
- **系统文件选择器**：浏览 Android 设备中任意 `.md` 文件
- **最近文件记忆**：智能记录最近打开的文件，方便快速返回

### 灵动编辑
- **所见即所得**：实时渲染的排版更符合移动端创作直觉
- **快捷工具栏**：常用 Markdown 标记一键插入
- **自适应布局**：竖屏横屏都能顺畅适配

### 主题个性化
- **暗夜模式**：深色主题深度优化对比度
- **多套配色**：纯黑 OLED、暖棕、冷灰等现代配色方案
- **自定义字体**：选择你喜欢的字体和字号

## Markdown 语法示例

### 文本格式

**粗体文本** 使用双星号包裹

*斜体文本* 使用单星号包裹

~~删除线~~ 使用双波浪线包裹

### 列表

无序列表：
- 第一项
- 第二项
- 第三项

有序列表：
1. 步骤一
2. 步骤二
3. 步骤三

### 代码

行内代码：`print('Hello, PureMD!')`

代码块：

```dart
void main() {
  runApp(const PureMDApp());
}
```
