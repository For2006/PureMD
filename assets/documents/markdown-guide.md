# Markdown 语法指南

Markdown 是一种轻量级标记语言，让你用纯文本格式编写结构化文档。

---

## 标题

使用 `#` 号标记标题，一个 `#` 是一级标题，逐级增加：

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

---

## 文本格式

| 样式 | 语法 | 示例 |
|------|------|------|
| **粗体** | `**文字**` | **粗体** |
| *斜体* | `*文字*` | *斜体* |
| ~~删除线~~ | `~~文字~~` | ~~删除线~~ |
| `行内代码` | \`代码\` | `print("hello")` |

## 列表

### 无序列表

```
- 项目一
- 项目二
  - 缩进项目
- 项目三
```

### 有序列表

```
1. 第一步
2. 第二步
3. 第三步
```

### 待办列表

```
- [ ] 未完成
- [x] 已完成
```

---

## 引用

使用 `>` 创建引用块：

```
> 这是一段引用文本。
> 可以跨越多行。
>
> > 支持嵌套引用。
```

实际效果：

> 这是一段引用文本。
> 可以跨越多行。
>
> > 支持嵌套引用。

### 行内代码

使用反引号包裹：\``code`\`

### 代码块

使用三个反引号包裹，可以指定语言以启用语法高亮：

\`\`\`python
def hello():
    print("Hello, PureMD!")
\`\`\`

\`\`\`dart
void main() {
  runApp(const PureMDApp());
}
\`\`\`


## 表格

```
| 左对齐 | 居中 | 右对齐 |
| :--- | :---: | ---: |
| 内容 | 内容 | 内容 |
| 内容 | 内容 | 内容 |
```

实际效果：

| 左对齐 | 居中 | 右对齐 |
| :--- | :---: | ---: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

---

## 转义字符

在特殊字符前加反斜杠 `\` 可以显示原本的标记符号：

```
\*不会被解析为斜体\*
\# 不会变成标题
```

---

## 块编辑器

PureMD 支持块编辑模式：

- **Enter** — 在当前块下方创建新块
- **退格** (空块时) — 删除当前块
- **点击左侧图标** — 切换块类型（正文、标题、引用、代码等）
